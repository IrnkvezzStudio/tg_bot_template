 ## установка
#
+ ### git clone github.com/IrnkvezzStudio/tg_bot_template.git
+ ### cd tg_bot_template
+ ### python -m venv env
+ ### pip install requerements.txt
+ ### Добавьте свой токен в файл config.py
+ ### python main.py
Если вы все сделали правильно, вы увидите 
> Bot started at (время)
